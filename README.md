# nodejs-reactjs-auth-multirole

# Cara menggunakan

## Database

- Pastikan Anda sudah membuat database dengan nama auth_db

## Backend

1. Masuk ke direktori backend

```Bash
cd backend
```

2. Install node_modules package

```Bash
npm install
```

3. Jalankan server

```Bash
node index
```

## Frontend

1. Masuk ke direktori frontend

```Bash
cd frontend
```

2. Install node_modules package

```Bash
npm install
```

3. Jalankan server

```Bash
npm start
```
